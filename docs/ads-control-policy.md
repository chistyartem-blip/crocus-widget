# Crocus Ads Control Policy

## Goal

Spend less, get more real bookings. Clicks are secondary. Every decision must answer:

- where did the client come from?
- how much did the client/lead cost?
- did they book?

## Campaign Roles

- PMax: Maps, brand, local coverage. Keep useful, but do not let it starve Search.
- Manicure Search: priority when it has conversions and bookable capacity.
- Pedicure Search: controlled test only until conversions prove it deserves more budget.

## Hard Rules

- Total daily budget cap: 20 EUR. Normal rescue range: 15-20 EUR for all active campaigns together.
- Scheduled runs are report-only. Live changes require manual dispatch.
- Never enable old paused junk automatically.
- Never add or enable broad keywords automatically.
- Never push a service without Altegio capacity.
- Never push a service that spent meaningful budget with zero conversions; use controlled testing instead.
- Do not optimize to soft/fake goals if booking quality is unclear.

## Allowed Automatic Actions

- Read Google Ads performance.
- Read Altegio hourly booking windows.
- Report master/service capacity.
- Suggest budget redistribution.
- Apply small budget/bid changes only through guarded manual dispatch.

## Current Rescue Logic

- Manicure gets priority because it has recent conversions and lost impression share.
- Pedicure gets controlled Monday/Tuesday push when Altegio has capacity, but broad Fußpflege stays capped because it can attract podology traffic.
- PMax is held at a low useful budget while Search is being rescued.

## What To Watch After Changes

- Search impressions within 30-90 minutes.
- Search lost impression share by rank and budget.
- Real booking conversions, not only clicks.
- Search terms that spend without bookings.
- Negative keywords that may block good local intent.
